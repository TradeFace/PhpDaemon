PhpDaemon 
==============


Run php daemonized


Installation
-------------

### for most recent release
add the following to composer.json
```json
    "require": {
        "tradeface/daemon":"master"
    }

```
```json
	"repositories": [
        {
            "type": "git",
            "url": "https://github.com/tradeface/daemon"
        }
    ],
```
